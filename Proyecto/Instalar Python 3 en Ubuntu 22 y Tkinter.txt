sudo apt update && sudo apt upgrade
sudo apt install software-properties-common –y
sudo add-apt-repository ppa:deadsnakes/ppa –y
sudo apt update
sudo apt install python3.8 –y
Verificamos la instalación y la versión
python3.8  --version
sudo apt install python3.8-dev –y
sudo apt install python3.8-venv –y
sudo apt install python3.8-distutils –y
sudo apt install python3.8-lib2to3 –y
sudo apt install python3.8-gdbm –y
sudo apt install python3.8-tk -y
sudo apt-get update
sudo apt-get install python3-tk
Instalar pip
Pip es el instalador de paquetes para Python. Con pip, podemos administrar fácilmente la instalación de paquetes de Python. Para instalar pip en su sistema ubuntu, simplemente ejecute este comando en la terminal.
sudo apt-get install python3-pip
Instalar la libreria OpenCV con pip
Después de la instalación de Python y pip, podemos instalar directamente la biblioteca OpenCV y comenzar a usarlos. Para instalar la biblioteca, debemos ingresar el comando dado en la terminal.
pip3 install opencv-python
Instalar matploplib
python3 -m install -U pip
python3 –m pip install –U matplotlib
Instalar pdf2image
